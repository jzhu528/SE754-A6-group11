# Java OOP Quiz Feature

This project implements a minimal Java OOP quiz feature for performance testing. Users start the quiz, choose a length (Quick 5, Standard 10, Full 15), answer multiple-choice questions, receive instant feedback, and view final score and review. 

The backend provides question retrieval APIs `GET /api/questions` and `POST /api/quiz/submit` endpoint that scores submitted answers and returns review data. Run the backend with `npm run dev` in `backend`, and the frontend with `npm run dev` in `frontend`.
